# My Portfolio Website

This is my personal portfolio website showcasing my projects, skills, and background. Built from scratch with HTML and CSS to keep it lightweight and fast.

**What's inside:**
- **Home**: Quick intro and hero section
- **About**: Who I am and what I do
- **Projects**: Featured work with links
- **Contact**: Ways to reach me

**Built with:** HTML5, CSS3, Flexbox/Grid for layout
**Goal:** Practice core web fundamentals without relying on libraries

**To view:** Just open `index.html` in any browser
